## 🚀 Как запустить макросы

1. Откройте Foundry VTT
2. Перейдите в **Macro** tab
3. Нажмите **Create Macro**
4. Вставьте код из нужного файла
5. Дайте макросу понятное имя
6. Нажмите **Execute** для запуска

---

## ⚠️ Важные замечания

- **macro-clear-cache.js** решает 90% проблем с rate limit
- Все макросы безопасны - не повредят систему
- Результаты выводятся в консоль (F12)
- Убедитесь что игра полностью загрузилась
- При проблемах с макросами используйте консольные версии

## 🎯 Ключевые команды для консоли

Если макросы не работают, выполните в консоли:
```javascript
// Очистка кеша
game.settings.set('blue-man-ai', 'aiProvider', 'gemini');

// Проверка настроек
console.log('Provider:', game.settings.get('blue-man-ai', 'aiProvider'));
console.log('API Key:', game.settings.get('blue-man-ai', 'apiKey') ? '✅' : '❌');
console.log('Models:', game.settings.get('blue-man-ai', 'aiModels'));
```
